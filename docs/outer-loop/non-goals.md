# Outer Loop Non-Goals

The outer loop is deliberately narrow. It is not:

- a TUI clone;
- a provider SDK;
- a credential manager;
- a daemon;
- a multi-agent scheduler;
- a transcript warehouse;
- a semantic proof engine;
- a replacement for human review;
- a license or terms workaround.

These non-goals keep phase two aligned with the baseline: one bounded prompt,
one fresh assistant session, inspectable validation, and operator-controlled
approval points.
