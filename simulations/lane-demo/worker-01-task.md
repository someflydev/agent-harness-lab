whether the fictional template documentation refresh can be represented with
the current manual lane structure.

- `simulations/lane-demo/orchestrator-brief.md`
- `simulations/lane-demo/lead-plan.md`
- `templates/lane/lane-brief.md`
- `templates/lane/worker-assignment.md`
- `templates/lane/lane-status.md`

## Expected Output

Write `worker-01-result.md` with changed files, evidence, assumptions,
blockers, and a concise lane-ready recommendation.

## Stop Conditions

Stop and escalate if the assignment requires editing real templates, if a
required input is missing, or if the result cannot be expressed as a local
artifact.
