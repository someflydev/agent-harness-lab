# Rejected Memory Decisions

This directory holds decision records for memory candidates that were rejected
or intentionally deferred.

Rejected records are useful when they prevent future sessions from reviving
speculative, stale, too-local, or noisy claims. They should explain the reason
for rejection and whether any temporary notes should be deleted.

Do not use this directory as a trash can for raw notes. If a candidate is simply
scratch context, leave it in ignored temporary state and prune it.
