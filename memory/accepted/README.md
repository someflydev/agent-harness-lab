# Accepted Memory Decisions

This directory holds decision records for memory candidates that were accepted
after review.

An accepted decision record should identify the candidate, evidence, reviewer,
durable update made, validation performed, and any cleanup. The durable memory
itself should usually live in a targeted doc, finding, runbook, template, test,
or script, not only in this directory.

Do not create accepted memory claims unless the claim is already validated by
repo evidence and the operator or approved process accepted it.
